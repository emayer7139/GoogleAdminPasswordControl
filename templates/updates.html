What’s New in ResetApp

📅 2025-04-23 Version 1.0.2

  • 🛠️ Admin Dashboard Overhaul: restructured into tabs with dedicated views:
    • Audit Logs — date filters & search 🔍
    • Login Logs — date filters & search 🔍
    • Users (Students) — live search, limited to /Students OU 👩‍🎓
    • Admins — manage list via live staff search 👥
    • Reset Requests — approve/deny daily limit increase requests ✅❌
  • 🔒 Admin Management: live-search staff-only OU when adding admins, with JS confirmation dialogs.
  • ⏱️ Password Reset Limits: configurable daily cap (default 5), plus new Request More workflow.
  • 🛡️ Staff-Only OAuth Gate: only allowed OU prefixes can sign in, plus DEV_LOGIN_BYPASS for local testing.
  • ⚙️ Error Handling: improved feedback, including custom 403 page and key-file validation with flash messages.

📅 2025-04-17 Version 1.0.1

  • 🐛 Fixed service account load error: added explicit JSON key check and UTF-8 validation.
  • 🎨 New Styling: refreshed layout with Bootstrap 5 cards and responsive improvements.
  • 🚀 Performance Improvements: optimized API calls and in-memory caching for logs.
