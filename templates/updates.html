📅 2025-04-23 Version 1.0.6

  • 🔄 Enhanced Bug Reporting: moved report button and known bugs to the top for quick access.
  • 🛠️ Admin Dashboard Overhaul: tabs for Audit, Login, Students, Admins, Reset Requests; date filters & search.
  • 🔒 Admin Management: live staff search limited to Staff OU, with confirmation dialogs.
  • ⏱️ Password Reset Limits: daily cap (default 5), + "Request More" workflow & approval.

  • ⚙️ Error Handling: custom 403 page, service-account JSON validation, flash feedback on key errors.

  • 📝 Reset Request Workflow: teachers can request extra resets; admins can approve/deny from the dashboard.
